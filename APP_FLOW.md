# App flow

- Home opens the dashboard with large cards for Recipes, Ingredients, Grocery / To-Buy List, and Pantry / At Home.
- Recipes: open the recipe list, create a new recipe, save it, and return to the recipe list.
- Ingredients: open the ingredient catalog, add an ingredient, and reuse it when creating recipes.
- Recipe form: choose from existing catalog ingredients when available, or type a recipe-only ingredient name.
